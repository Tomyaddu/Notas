Los servidores ProLiant todos vienen incluido con iLO y Silicon Root of Trust. La familia consta de:

- MicroServers: Servidores chicos pensados para oficinas chicas o casas
- ML Family: Servidores en forma de torre (Como una PC), pensado para oficinas remotas.
- DL Family: Servidores para racks
- RL Family: Servidores pensados para trabajos en la nube, vienen con procesadores Amperes
- Synergy: Soluciones Blade (Servidores de Hoja)
- XD Family: Servidores para racks, para soluciones potentes de computo y para uso de IA

Nos comenta que iLO nos permite remotamente monitorear, administrar y hacer troubleshooting a un servidor desde donde sea, reduciendo el tiempo no operacional.
